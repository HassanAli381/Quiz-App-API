Common Status Codes Explained
These codes are grouped into categories, making them easier to understand: 
2xx: Successful Responses
200 OK
: The request was successful, and the server responded as expected. 
201 Created
: The request was successful, and a new resource was created on the server. 
204 No Content
: The request was successful, but there is no content to send back to the client. 
3xx: Redirection Messages 
301 Moved Permanently
: The requested URL has permanently moved to a new location. 
302 Found (Temporary Redirect)
: The requested resource has been temporarily moved to a different URL. 
4xx: Client Errors
400 Bad Request
: The server could not understand the request due to invalid syntax or a client-side error. 
401 Unauthorized
: The client must authenticate itself to access the resource. 
403 Forbidden
: The server understood the request but refuses to authorize it, often due to permission issues. 
404 Not Found
: The server could not find the requested resource. 
5xx: Server Errors
500 Internal Server Error
: An unexpected error occurred on the server that prevented it from fulfilling the request. 
503 Service Unavailable
